Portal Gun

The Portal Gun is an unused item which shoots a portal that can teleport you to another portal. It has the same texture as the Phaser though it is incredibly buggy. The gun will often crash the user upon use though can create two portals in which a duck can step slightly through one and see part of itself through the other end. The duck will not be able to pass through these portals.

It's a reference to the game "Portal" which has a similar weapon that also can create portals.
